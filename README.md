An android weather application which demonstrates the following features:

• Platform : Android

• Language Used : Kotlin

• Architecture: MVVM

• Unit testcases : Junit and Mockito

• UI test cases : Espresso

• Retrofit2

• RxJava

• LiveData

• Singleton Class

• Latitude & Longitude using Geolocation.
